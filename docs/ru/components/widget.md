# Widget

## Краткое описание
Приложение на Angular/Ionic с интерактивной панелью виджетов.

## Основные виджеты
- **Погода** - данные с API wttr.in
- **Часы** - текущее время с автообновлением
- **Фаза Луны** - расчет через MoonService

## Дополнительные виджеты
- **Заметки** - текстовые блоки
- **Счетчик** - инкремент/декремент значений
- **Секундомер** - с кругами и управлением
- **Текст** - простые текстовые виджеты

## Особенности
- Автосохранение в localStorage
- Ограничение по типам виджетов
- Управление через ActionSheet
- Интеграция с внешними API
- Поддержка Ionic UI компонентов

## Возможные улучшения
- Перетаскивание виджетов (Drag & Drop)
- Изменение размера виджетов
- Темная/светлая тема
- Погода для разных городов
- Календарь виджет
- Котировки акций/криптовалют
- Погода по геолокации
