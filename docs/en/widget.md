# Widget

## Brief Description
An Angular/Ionic app with an interactive widget panel.

## Main Widgets
- **Weather** - data from the wttr.in API
- **Clock** - current time with automatic update
- **Moon Phase** - calculated via MoonService

## Additional Widgets
- **Notes** - text blocks
- **Counter** - increment/decrement values
- **Stopwatch** - with circles and controls
- **Text** - simple text widgets

## Features
- Auto-save to localStorage
- Limits on widget types
- Control via ActionSheet
- Integration with external APIs
- Support for Ionic UI components
